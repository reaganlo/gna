In order to run the sample:
1. put gna.lib and gna.dll into GNA\[Win32|x64] directory
2. put gna-api.h, gna-api-status.h, gna-api-types-gmm.h, gna-api-types-xnn.h into GNA\include directory
3. install Redistributable Libraries for Intel(R) Parallel Studio XE 2015 if necessary
   https://software.intel.com/en-us/articles/redistributables-for-intel-parallel-studio-xe-2015-composer-edition-for-windows
4. install Redistributable Libraries for Microsoft Visual Studio 2015 if necessary 
   https://www.microsoft.com/en-us/download/details.aspx?id=48145
5. load the gna-samples.sln in Visual Studio 2015 and build the solution
6. step through the sample code in Visual Studio 2015
